# Backend
## Installation
1. Clone the repo
```sh
git clone https://github.com/OAMK-VPN/CustomerApp_frontend.git
```

2. Update properties in application.properties `(server/src/main/resources/)`

## Usage
1. Run the server
```sh
cd server
mvn spring-boot:run
```
